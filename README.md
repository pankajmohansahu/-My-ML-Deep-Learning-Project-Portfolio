## 🧠 About This Portfolio

Build real-world, end-to-end ML & Deep Learning systems — from raw data ingestion to deployed inference services. Projects span **NLP**, **speech processing**, **sentiment analysis**, **time series forecasting**, and **agentic AI systems**, all implemented with production-grade practices: robust preprocessing, fine-tuning LLMs, experiment tracking, XAI explainability, and cloud-based deployment.

---

## 🔄 ML / DL Workflow

| Stage | Details |
|-------|---------|
| **Problem Framing** | Define objectives, metrics, and baseline expectations |
| **Data Collection & Preprocessing** | Cleaning, tokenization, feature engineering, train/val/test splitting |
| **Model Development** | Fine-tuning LLMs, training from scratch, or classical ML pipelines |
| **Experiment Tracking** | Log runs, compare metrics, version models |
| **Evaluation** | Accuracy, RMSE, F1, leaderboard ranking, human eval |
| **Deployment** | HuggingFace Spaces, GCP Vertex AI, Gradio demos |

## 🚀 Projects

| Project | Description | Skills |
|---------|-------------|--------|
| [Mental Health Speech Detection](https://github.com/pankajmohansahu/-My-ML-Deep-Learning-Project-Portfolio) | ML system to detect depression and emotional state from voice using MFCC features and Whisper embeddings. Built as an IIT Madras AI & ML Lab capstone project combining speech processing with mental health classification. | Whisper, MFCC, PyTorch, Speech Processing, HuggingFace |
| [Multilingual Speech Recognition](https://github.com/pankajmohansahu/-My-ML-Deep-Learning-Project-Portfolio) | Multilingual speech recognition system using Whisper large-v3. Includes fused prompt hallucination fix, adaptive beam search, and language-specific fine-tuning across multiple Indian languages. | Whisper, HuggingFace, PyTorch, NLP, Multilingual |
| [Multilingual Sentiment — India](https://github.com/pankajmohansahu/-My-ML-Deep-Learning-Project-Portfolio) | Fine-tuned Google Gemma 2-2B using LoRA on 9 Indian languages for binary sentiment classification. Achieved **90% accuracy** on low-resource Indian language datasets using prompt engineering. | Gemma 2, LoRA, PEFT, HuggingFace, Prompt Engineering, GCP |
| [Emoji Emotion Classification](https://github.com/pankajmohansahu/-My-ML-Deep-Learning-Project-Portfolio) | Fine-tuned Gemma 2-2B with QLoRA for 8-class emotion classification in low-resource Indian languages including Tamil, Kannada, and Malayalam. IIT Madras BSMS Lab 2025. | Gemma 2, QLoRA, Multilingual NLP, HuggingFace, PyTorch |
| [Stock Movement Prediction](https://github.com/pankajmohansahu/-My-ML-Deep-Learning-Project-Portfolio) | Ranked **#1** on the IIT Madras BSMS Lab Kaggle competition for predicting daily stock closing prices. Uses XGBoost with a residual stacking strategy combining MLP for final predictions. | XGBoost, MLP, Time Series, Feature Engineering, Kaggle |
| [News Summarization & Sentiment](https://github.com/pankajmohansahu/-My-ML-Deep-Learning-Project-Portfolio) | LangGraph-based AI Agent that autonomously fetches news articles, summarizes them using Groq (Llama 3.1 8B LLM), and performs sentiment analysis using a rule-based tool pipeline. | LangGraph, Groq, LLaMA 3.1, Agentic AI, NLP, Python |
| [House Price Prediction](https://github.com/pankajmohansahu/-My-ML-Deep-Learning-Project-Portfolio) | ML regression model for predicting California housing prices using Gradient Boosting. Achieved **0.48 RMSE** with automated feature engineering and hyperparameter tuning pipeline. | Gradient Boosting, sklearn, Feature Engineering, Regression |
| [Maritime Survivor Detection](https://github.com/pankajmohansahu/-My-ML-Deep-Learning-Project-Portfolio) | ML model for maritime accident survivor prediction using Random Forest classifier. Built for the IIT Madras Data Science Competition. | Random Forest, sklearn, Classification, Python |
